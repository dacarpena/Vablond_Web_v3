# Vablond - Landing Page

Landing page estática para Vablond S.L. - Gestión Eficiente de Residuos.

## Tecnologías

- **HTML5** - Estructura semántica
- **Tailwind CSS** (CDN) - Estilos utilitarios
- **JavaScript** (Vanilla) - Interacciones

## Estructura del proyecto

```
web_vablond_v3/
├── index.html    # Página principal
├── styles.css    # Estilos personalizados
├── main.js       # JavaScript para interacciones
└── README.md     # Este archivo
```

## Cómo usar

### Opción 1: Abrir directamente
Simplemente abre `index.html` en tu navegador.

### Opción 2: Servidor local
Para desarrollo con live reload, puedes usar cualquier servidor estático:

```bash
# Con Python
python -m http.server 8000

# Con Node.js (npx)
npx serve .

# Con PHP
php -S localhost:8000
```

## Despliegue

Esta landing page estática puede desplegarse en cualquier servicio de hosting:

- **GitHub Pages** - Gratis, simplemente sube los archivos
- **Netlify** - Arrastra y suelta la carpeta
- **Vercel** - Importa el repositorio
- **Cualquier hosting compartido** - Sube via FTP

## Características

- Diseño responsive (mobile-first)
- Animaciones suaves
- Formulario de contacto
- Sin dependencias de build
- Carga ultra-rápida
- SEO básico incluido

## Personalización

- **Colores**: Modifica las variables en `tailwind.config` dentro de `index.html`
- **Estilos**: Ajusta `styles.css` para estilos personalizados
- **Contenido**: Edita directamente el HTML en `index.html`
- **Interacciones**: Modifica `main.js` para comportamiento JavaScript
